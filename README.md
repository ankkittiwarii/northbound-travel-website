# NorthBound Travel Website

NorthBound is a travel and tourism website that allows users to explore travel destinations, view travel packages, and book trips online.

## 🌐 Technologies Used

* HTML
* CSS
* JavaScript
* PHP (Backend)
* MySQL (Database)

## 📂 Project Structure

```
northbound-travel-website
│
├── index.html
├── pages
│   ├── booking.html
│   ├── contact.html
│   ├── login.html
│   └── signup.html
│
├── assets
│   ├── css
│   ├── js
│   └── images
│
├── backend
│   ├── auth
│   ├── booking
│   └── config
│
└── database
```

## ✨ Features

* User Signup and Login
* Travel Packages
* Destination Gallery
* Online Booking System
* Contact Form
